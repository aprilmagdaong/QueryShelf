About QueryShelf

QueryShelf is a modern and efficient library management system developed for the Local Government of Daraga to simplify and improve daily library operations.

The system provides complete CRUD (Create, Read, Update, Delete) functionality for managing books, borrowers, and library records with accuracy and ease. Borrowers can conveniently browse library materials and check the availability of books, while authorized personnel such as the IT Staff, Library Clerk, and Librarian are given secure access to the main system for managing transactions and records.

QueryShelf also features an automated circulation system that monitors borrowed books, tracks due dates, and automatically calculates fines for overdue materials, helping the library maintain organized, reliable, and efficient services.
